# Fonts go here

Place any `.ttf`, `.otf`, or `.ttc` font file in this directory.
They will automatically appear in the **font_name** dropdown of the
Speech Bubble node.

Korean fonts that work well:
- NanumGothic.ttf  (무료, 네이버 나눔 폰트)
- NanumMyeongjo.ttf
- GmarketSansMedium.otf

⚠️ Fonts are NOT included here due to copyright/licensing.
   Please download and copy the font files yourself.
